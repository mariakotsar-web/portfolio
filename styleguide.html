Accessibility

WCAG 2.1 AA compliant. Focus visible outlines, reduced motion support, semantic landmarks, keyboard-navigable overlay with focus trap.

Color Contrast

Primary text AAA 15.4:1
Secondary text AA 7.5:1
Accent cyan AAA 13.1:1

Focus Indicators

Focused Focused

outline-offset: 3px
Applied via :focus-visible

Skip Link

Reduced Motion

@media (prefers-reduced-motion: reduce)
• All animation-duration → 0.01ms
• All transition-duration → 0.01ms
• Scroll-snap disabled
• .sr elements forced visible
• Hero animations removed
• scroll-behavior: auto

Touch Targets

Min touch target: 44×44px
Applied to: footer links, close button
Interactive tiles: full-panel hit area
Buttons: 16px vert / 36px horiz padding

Semantic Structure

Landmarks: <main>, <footer>, <nav>, <section>
aria-hidden on decorative elements
aria-label on icon-only links
role="dialog" + aria-modal on overlay
Focus trap with Escape key close
